US Chocolate Company Performance Dashboard

This Power BI project presents an executive-level performance analysis of a US-based chocolate company. The dashboard provides insights into sales, profit, customer trends, and product performance across multiple years, supporting strategic and operational decision-making.

Project Overview
The dashboard is designed with a multi-page layout, including Home, Executive Overview, Sales Overview, and Profit Overview sections. It enables stakeholders to quickly assess business performance, identify trends, and evaluate key drivers affecting revenue and profitability.

Key Insights Covered
Sales and profit performance across years (2023 and 2024)
Customer volume and category-level analysis
Profit margin trends and operational efficiency insights
Top-performing and declining products
Regional profit distribution using geographic visualization
Quarterly sales and profit trend analysis

Dashboard Structure
Home page with navigation and high-level context
Executive Overview summarizing customers, sales, profit, and margin
Sales Overview highlighting revenue trends and top products
Profit Overview focusing on profitability trends and margin performance

Tools and Techniques Used
Power BI Desktop
DAX measures for sales, profit, margin, and trend analysis
Data modeling using calendar, product, location, and shipment tables
Interactive slicers for year and category selection
Custom visuals and report navigation for executive storytelling

Purpose
This project was created to demonstrate business intelligence, data modeling, and data storytelling skills using Power BI. It is intended for portfolio presentation and showcases the ability to translate raw business data into meaningful executive insights.
